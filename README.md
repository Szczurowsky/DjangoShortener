## Features
- Shortener
- Save to database
- Protected from doubling URL's
## Setup
- Clone repository `https://github.com/Szczurowsky/djangoShortener`
- Place into Django directory
- Add url to app in main url file. `path('', include('shortener.urls'))`
- Enjoy
